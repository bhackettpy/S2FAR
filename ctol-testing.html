Conventional Takeoff and Landing (CTOL) Flight Test Validation

This project builds on prior VTOL-focused Trajectory Control System (TCS) research, adapting the architecture to support short takeoff and landing (STOL) and conventional takeoff and landing (CTOL) operations. The TCS provides configuration-invariant control laws and trajectory tracking, reducing pilot workload while maintaining precise path control. Development and validation integrate high-fidelity simulation with subscale flight testing using a 2.1 m wingspan Carbon-Z Cessna 150T platform. Ongoing tests assess handling qualities, control robustness, and performance across the fixed-wing flight envelope, leveraging the same control architecture proven on multiple VTOL configurations.
